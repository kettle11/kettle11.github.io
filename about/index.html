About Ian

I aim to create things that help people be happy, creative, and productive.

From an early age I spent chunks of my childhood living on a sailboat, which gave me plenty of time to learn to code in NetLogo. Mostly I made procedural art and tiny games.

Some years later I worked on the HoloLens shell team at Microsoft.

Since then I've left Microsoft to independently create a 3D design tool for virtual reality, which I'm working on now.

  • Blueberry Bounce
  • Noah's Help

And made some Rust libraries:
