Welcome to Wabi!

Wabi is a work in progress. Check back later to learn more!

As I work on Wabi I've been updating this page to test out Git integration. Today I made Git integration work super seamlessly! Git isn't hard, but often times when you're in a creative or flow state you don't want to remember exact git commands. Wabi handles all that Git stuff for you so you just type "Publish" and your work is online!

To update your website with Wabi run just one command: "publish". Your new website will be online in seconds.

Wabi is super fast!
