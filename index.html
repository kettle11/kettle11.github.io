Wabi: Elegantly Build Websites


Wabi is very simple.

1. Download Wabi
2. Put Wabi somewhere
3. Click on it to begin

Now you have a static website generated with a template you can edit and a folder you can put posts into.

When you're ready to upload your website just type "publish" and Wabi will upload any changes to your Git repository.

Have fun!
